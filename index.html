ChitLAN

Secure peer-to-peer group chat, file sharing, voice chat, and screen sharing for local networks.

ChitLAN is a browser-based LAN communication app that allows users to chat, share files, talk over voice, and broadcast their screen instantly within a local network. No servers, no cloud storage, just fast private communication.

🔒 Private & Secure

No servers. No cloud. Communication happens directly between peers.

Optimized for low latency within local networks.

📁 Large File Sharing

Send images, videos, and large files with progress tracking.

🖥️ Screen Sharing

Broadcast your screen live to connected users in the room.
