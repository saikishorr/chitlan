# 💬 ChitLAN – Real-Time Collaboration on Your Local Network

**Latest Version:** **v2.0.0 – Real-Time Collaboration Edition**

**ChitLAN** is a lightweight, browser-based **peer-to-peer communication platform** designed for Local Area Networks (LAN). It combines group messaging, voice chat, live screen sharing, and high-speed file transfers into a single application—all without requiring servers, cloud services, or user accounts.

Built entirely with **WebRTC** and **PeerJS**, ChitLAN enables secure, direct communication between devices on the same network.

# ✨ Features

## 💬 Real-Time Chat

- ✅ Peer-to-peer group chat powered by WebRTC (PeerJS)
- 👥 Supports up to **10 users** per room
- 💬 Clean left/right message layout
- ⌨️ Live typing indicators
- 🕒 Message timestamps
- 🔔 Notification sounds for new messages
- 📋 One-click Room ID copy
- 💾 Automatic chat history persistence
- 🌙 Dark Mode support
- 📱 Fully responsive design

---

## 🎤 Voice Chat

- 🎙️ Real-time peer-to-peer voice communication
- 🔇 Mute / Unmute microphone
- 🚪 Join or leave voice sessions anytime
- 👥 Live voice participant list
- 🎧 Browser-based audio streaming
- 🎤 Automatic microphone permission handling

---

## 🖥️ Screen Sharing

- Share your screen in real time
- Built-in screen viewer
- One-click start and stop controls
- Automatic notifications when screen sharing begins or ends
- Peer-to-peer streaming using WebRTC

---

## 📁 Advanced File Sharing

- 📤 Direct peer-to-peer file transfer
- 📦 Supports files up to **100 GB**
- ⚡ Chunk-based transfer for improved reliability
- 📊 Live transfer progress bars
- 🚀 Transfer speed display
- 🖼️ Image preview inside chat
- 📥 Download links for received files
- ✔️ Automatic transfer completion confirmation
- 🔁 File acknowledgement system

---

## 🌐 Network Features

- 🌍 Automatic Public IP detection
- 🔄 Automatic PeerJS reconnection
- 📡 No server required for messaging or file transfers
- 🏠 Designed for Local Area Networks (LAN)

---

## 📲 Progressive Web App (PWA)

- Install directly from supported browsers
- Works like a native desktop application
- Service Worker support
- GitHub Pages compatible

---

## 🔒 Reliability & Performance

- HTML escaping for secure messages
- Automatic cleanup of media streams
- Improved connection recovery
- Optimized large file transfers
- Duplicate file chunk protection
- Better memory management
- Faster rendering and smoother scrolling

# 📦 Getting Started

### Clone the repository

```bash
git clone https://github.com/saikishorr/chitlan.git
cd chitlan
```

Open `app.html` in your browser or deploy the project using **GitHub Pages**.
